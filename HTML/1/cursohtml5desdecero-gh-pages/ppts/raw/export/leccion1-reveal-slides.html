Alojamiento y servidores HTTP

Definiciones

  • Servidor web HTTP (o simplemente: servidor web o servidor HTTP): es programa que se encarga principalmente de las comunicaciones con el navegador.

  • Un alojamiento web (web hosting o hosting): es un ordenador conectado a Internet. Comúnmente también se le llama: servidor.

Servidores web y empresas de hosting

  • Empresas de hosting: 1and1, AWS, Linode, etc. (ver más)

  • Servidores web: Apache, Nginx y IIS, etc. (ver más)

Navegadores

Definición

Un navegador es un programa complejo que entiende y transforma el código que recibe desde un servidor en algo entendible para un humano (una página web, imágenes, etc).

Responsabilidades

  • Integridad

  • Seguridad

  • Optimización

Peticiones HTTP

Tipos de peticiones

  • GET: para solicitar información.

  • POST: para enviar información.

Mensajes de respuesta

  • 200: se ha encontrado correctamente el fichero/recurso.

  • 403: no tenemos permiso para acceder al fichero/recurso.

  • 404: el fichero/recurso que le hemos solicitado no se ha podido encontrar.
