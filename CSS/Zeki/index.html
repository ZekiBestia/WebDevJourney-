Ambientes virtuales con Python

El 16 de Enero del 2024 se llevo acabo un evento de la comunidad de Python en la ciudad de México, en las instalaciones de Wizeline, en la cual se hablo de los ambientes virtuales y como se pueden utilizar para el desarrollo de aplicaciones en Python.

El evento fue impartido por David Sol y organizado por Charly Román

¿Qué son los ambientes virtuales?

Los ambientes virtuales son entornos aislados que se crean para gestionar las dependencias y bibliotecas especificas de un proyecto. Esto permite trabajar en un entorno de desarrollo limpio evitando conflictos de dependencias.

¿Qué problemas resuelve?

  • Conflicto de dependencias.
    • Diferentes proyectos requieren de versiones especificas versión especifica de una biblioteca o paquetes. Si no tuvieramos ambientes virtuales estas dependecias entrariar en conflicto si se instalan de manera global.
  • Versiones de Python
    • Algunos proyectos requieren de una versión especifica de Python, por lo que se puede crear un ambiente virtual con la versión de Python que se requiera.
